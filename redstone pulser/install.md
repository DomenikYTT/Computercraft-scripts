# Redstone Repulser
Redstone Repulser is a script for ComputerCraft that converts Redstone signals to different lengths; you can configure settings such as the Redstone input and output locations. You simply need to install the script as described in the instructions below.



## Setup

1. you need an Advanced Computer and the CC: Tweaked Mod and 2 redstone dust (or any other redstone block)

2. you place the Advanced Computer and open it, and type **pastebin get gbi1yWk** into the console.

3. third step: you can edit the config with **edit config.lua**, there you can set the input and output side, the sides are as an comment at the top (--.... are the comments.)

4. then you save the config with pressing **CTRL** or in german with **STRG** and then click on the save then press again the key and click on save.

5. type **startup** in the console and you are done, now you can now use it with you redstone machines!
