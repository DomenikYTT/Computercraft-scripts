# Energy Core (Energy Battery from Draconic Evolution)


## What does it do?

### it simply displays:
- Tier
- Capacity
- Stored
- Input/output for the gates and for the core itself


## How-To-Use

1. you place the sdvanced computer infront of the **Energy Core Stabilizer**
2. you type **pastebin get EekiS1US install**
3. you connect 1 or more monitors in your network
4. type **install**
5. you can now edit the config with **edit config.lua** and save it.
6. type **startup**
7. now you are finished and the monitors should display its tier, Stored, Capacity and more.
